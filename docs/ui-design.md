# UI 设计 — Easy Learn English（macOS）

## 布局（3 栏）
- **左侧**：媒体库
  - 导入按钮
  - 媒体列表（音频/视频）
  - 状态徽标：`Transcribed`、`Pending`、`Error`
- **中间**：播放器 + 字幕
  - 顶部：视频/音频播放器
  - 中部：当前片段高亮
  - 底部：同步滚动字幕列表
  - 单词可点击，`Shift` 选择范围
- **右侧**：单词详情 + 生词本
  - 详情卡片
    - 选中的词/短语
    - 英文释义
    - 中文翻译
    - 保存/移除按钮
  - 生词本列表（滚动）

## 交互细节
- 单词点击选中。
- `Shift+点击` 可在同一字幕行中选择连续单词。
- 字幕随播放自动滚动。
- 选择变化即时更新右侧翻译。
- 保存后出现在生词本列表。

## 视觉风格
- 使用系统字体，保持 macOS 原生感。
- 中性色背景；当前片段与选中词使用强调色。
- 面板之间使用细分隔线。

## 可访问性
- 支持动态字体大小。
- 字幕列表支持键盘导航。
- 明确的选中状态。

## 设置（弹出或独立页）
- 转写提供商选择。
- API Key 填写。
- 语言设置（默认：英语）。
